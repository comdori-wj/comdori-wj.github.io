# ⏱️ 밀리초타이머 (Millisecond Timer) Web

iOS 앱 [밀리초타이머](https://apps.apple.com/app/id6462732415)의 공식 랜딩 페이지 프로젝트입니다.  
사용자에게 앱의 주요 기능을 소개하고 App Store 다운로드를 유도하며, 다국어 및 테마 설정을 지원하는 반응형 웹사이트입니다.

## 🚀 배포 주소 (Demo)
👉 **[https://comdori-wj.github.io/](https://comdori-wj.github.io/)**

---

## ✨ 주요 기능

### 1. 🌍 다국어 지원 (i18n)
- 한국어(KO) 및 영어(EN)를 완벽하게 지원합니다.
- 접속 시 사용자의 **시스템 언어를 자동으로 감지**하여 표시합니다.
- 사용자가 선택한 언어 설정은 브라우저에 저장되어 재방문 시 유지됩니다.
- 언어 변경 시 텍스트뿐만 아니라 **앱 스크린샷 이미지**와 **App Store 다운로드 배지**도 해당 언어에 맞게 즉시 전환됩니다.

### 2. 🎨 테마 시스템 (Dark/Light Mode)
- **시스템 설정(자동)**, **라이트 모드**, **다크 모드**를 지원합니다.
- CSS Variables를 활용하여 테마 변경 시 부드러운 전환 효과를 제공합니다.
- 히어로 섹션의 배경과 아이콘 쉐도우 등 세세한 부분까지 테마에 반응합니다.

### 3. 📱 반응형 디자인
- 데스크탑, 태블릿, 모바일 등 모든 디바이스에서 최적화된 레이아웃을 제공합니다.
- 모바일 환경에서는 헤더 컨트롤이 한 줄로 정렬되어 공간 효율성을 극대화합니다.
- 스크린샷 갤러리는 마우스 휠(데스크탑)과 터치 스와이프(모바일)를 모두 지원합니다.

### 4. 📊 방문자 통계
- 외부 API를 연동하여 실제 방문자 수(오늘/누적)를 카운트합니다.
- API 호출 실패 시 로컬 스토리지 기반의 백업 로직이 작동하여 끊김 없는 사용자 경험을 제공합니다.

## 🛠 기술 스택

- **HTML5**: 시맨틱 마크업 및 SEO 최적화 (Open Graph 태그 포함).
- **CSS3**: Vanilla CSS 사용 (Flexbox, Grid, CSS Variables). 별도의 프레임워크 없이 가볍고 빠른 성능 보장.
- **JavaScript (ES6+)**: 외부 라이브러리 의존성 없이 순수 자바스크립트로 구현.
